Comment se créer un compte :

  • Première étape

    Rendez-vous sur le site https://lightshope.org/

  • Deuxième étape

    Créez vous un compte en allant à droite de l'écran dans la section "Create an account".

  • Troisième étape

    Une fois la création de compte effectuée, vous allez recevoir un email de confirmation, cette étape ne devrait pas prendre plus de 5 minutes

  • Quatrième étape

    Une fois l'email de confirmation reçu, cliquez simplement sur le lien intitulé "activate your account" pour activer votre compte. Une fois cela terminé, votre compte est crée ! Félicitations !

Comment télécharger le jeu :

Sous windows

Il y a plusieurs moyens de trouver le jeu :
  1. Recherchez sur google une version wow 1.12
  2. LIEN DIRECT en 1 partie (5.20 Go) Version Française
  3. LIEN DIRECT en 1 partie (5.20 Go) Version Anglaise

Sous Mac

Les liens pour le client mac sont plus difficiles à trouver
  1. Recherchez sur google une version wow1.12 mac
  2. LIEN DIRECT en 1 partie (5.20 Go) Version Française MAC
  3. Attention pour jouer sur mac, vous devez télécharger les deux versions du jeu, c'est à dire la version Windows et la version Mac

Voici le site sur lequel je vais chercher les liens

Comment installer le jeu

Sous windows

  1. Extraire le jeu une fois qu'il est téléchargé
  2. Copiez le jeu sur votre bureau
  3. Changez la reamlist, vous devez mettre celle du serveur light's hope (set realmlist logon.lightshope.org)
  4. Lancez le jeu en cliquant sur Wow.exe

Sous Mac

  1. Une fois les deux versions du jeu téléchargées, dézippez ces deux fichiers
  2. Changez la reamlist, vous devez mettre celle du serveur light's hope (set realmlist logon.lightshope.org) dans le fichier windows
  3. Copiez l'application du dossier mac et mettez la dans le fichier windows, lancez l'application pour commencer à jouer
